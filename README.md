# chat-lion (under development)

  <img align="right" width="260" src="resources/logo/lion.png">

English | [简体中文]()

chat lion 是一个采用现代 Go 技术栈构建即时通讯后端系统，使用 Gin、GORM、Redis、WebSocket 和 Kafka，实现了一个功能丰富的聊天应用。

[documentation]()

[fronted project]()

## ✨ 特点

- 消息分片

- 支持分布式部署

- JWT authentication

- 功能:
     - 好友管理✅
     - 单聊和群聊✅
     - 文字/语音/文件✅
     - 语音聊天❌
     - AI聊天❌
     - 此刻❌
     - 聊天记录备份❌
     - 实时语音转录+情感分析❌
     - 跨语言无障碍沟通❌
     - 对话摘要与决策提炼❌
